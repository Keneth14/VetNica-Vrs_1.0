VetNica es un sistema de gestión veterinaria que permite registrar animales, llevar su historial médico, gestionar alertas y controlar la facturación de manera fácil y ordenada; para usar el sistema solo debe abrir la carpeta del programa y hacer doble clic en el archivo START, luego el sistema se abrirá automáticamente en su computadora y podrá comenzar a trabajar desde el navegador sin necesidad de configuraciones adicionales, ya que todo está listo para funcionar en modo local dentro del equipo.
